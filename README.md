# Verilog Processor Datapath

This repository documents my journey of building a processor datapath from scratch using Verilog HDL.

The project includes parameterized RTL modules, simulation using Icarus Verilog and GTKWave, and synthesis using Yosys.

# Completed Modules

 -8-bit Parameterized ALU
 -8×8 Register File (2 Read, 1 Write Port)
 -ALU + Register File Integration


# Tools Used

- Verilog HDL
- Icarus Verilog
- GTKWave
- Yosys
